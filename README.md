# Bondi Website

A modern responsive website built using HTML and CSS.

## 🚀 Features
- Fully responsive design for all screen sizes
- Clean and modern UI
- Multiple sections:
  - Header & Navigation
  - Hero section
  - Services
  - Portfolio
  - About
  - Team
  - Blog
  - Footer
- Hover effects and smooth UI interactions
- Custom CSS variables for easy customization

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox + Media Queries)
- Font Awesome Icons

## 📱 Responsive Design
The website is fully responsive and works smoothly on:
- Mobile devices
- Tablets
- Laptops
- Large screens
